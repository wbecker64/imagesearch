FreeCodeCamp : Image Search Abstraction Layer

User stories:

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Example :

https://look4img.herokuapp.com/api/imagesearch/flowers?offset=2

Output example:

[ { "url":"http://images.shopflowers.net/images/products/VL_711250.jpg","snippet":"Sarasota Florists - Flowers in Sarasota FL - Flowers By Fudgie On ...", "thumbnail":"https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcSgpeavAeJQLPiP4PcUEsz8trNkcw3zn8qsf2JKF1mrNg2TlB2MBXIau3w", "context":"http://www.flowersbyfudgie.com/" } ... ]

Usage:

https://look4img.herokuapp.com/api/latest/imagesearch

Output example:

[ { "term":"flowers","when":"2018-5-10 10:26:36" } ... ]
